# WebInstituto
Para iniciar el proyecto debemos ejecutar el comando de npm start, esto nos dará indicación de que ya se encuentra corriendo en servicio nuestra app

# Verificar DB
Cada uno deberá dirigirse al archivo .env para cambiar el valor de la ruta, de esta manera, podremos acceder a la DB y crear nuestra propia conexión
